# LOCK🔒D

**Platform:** iOS 📱  
**Team Name:** Healthiest People 💪

LOCKD is an iOS productivity app that helps users *lock in* while studying. By combining timelapse recording, social accountability, and competition, LOCKD encourages focus, consistency, and healthy productivity habits.

## ❓ Problem

Staying motivated and consistent while studying is hard. Many productivity apps are easy to ignore and don’t provide real accountability.

LOCKD tackles this by:
- 📹 Requiring timelapse recordings to reduce phone distractions  
- 🧑‍🤝‍🧑 Using social pressure and visibility to encourage consistency  
- 🏆 Adding competition through leaderboards  

---

## 🚀 Core Features (MVP)

- 🎥 **Timelapse Study Recording**  
  Record yourself studying to stay focused and avoid distractions.

- 📰 **Community Feed**  
  Upload study sessions, view others’ posts, and interact through comments and reactions.

- 👤 **User Profiles**  
  Track study history, streaks, XP, and personal statistics.

- 🏆 **Leaderboards**  
  Rank users by productivity to encourage competition and motivation.

## 🏗️ Architecture Overview

- 📱 iOS app built with SwiftUI  
- 🔥 Firebase backend for authentication, data, and media storage  
- 🔁 Communication via Firebase SDKs  

---

## 🔐 Privacy & Security

- Camera access requires explicit user permission  
- User accounts handled securely through Firebase Auth  
- Minimal collection of personal information
